Le codage d'affichage -- WEB Le codage de traitement -- APPLICATIONS
Le codage d'affichage est le codage qui permet d'afficher des informations sur un écran. Il est composé de balises HTML et de feuilles de style CSS. Le codage de traitement est le codage qui permet de traiter des informations. Il est composé de langages de programmation comme le PHP , le JavaScript , le C- , le C++ , le Java , le Python , etc.
